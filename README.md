 🦅 WildEye — AI Wildlife Monitoring System

A Final Year Project that uses **YOLOv8n** deep learning model to automatically detect and identify wild animals from images and video footage using computer vision.

 🛠️ Technologies Used
- Python
- YOLOv8n (Ultralytics)
- OpenCV
- React (Frontend)
- Node.js

 🚀 How to Run

Backend:
```bash
pip install -r requirements.txt
python detect.py
```

Frontend:
```bash
npm install
npm start
```
